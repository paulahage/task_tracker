### Running the project:

```
$ git clone git@github.com:paulahage/task_tracker.git
$ cd website
$ npm install
```
#### Run the server:

```
$ cd website
$ npm run server
$ npm start or ng serve
```
- You can check the API at http://localhost:5000/

#### Run the project:

```
$ cd website
```
